MemoryGame
----------
A simple memory game in which you should flap 16 cards that lay face downwards in pair as fast as possible.
----------
This program uses libsx for a more convenient X windows programming.
Source codes of libsx in provided in the repository.
You should have development files (headers and static libraries) for libsx installed on your system to build this program.
See https://web.physics.wustl.edu/marty/libsx/libsx.html for more details.
